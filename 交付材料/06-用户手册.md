## 1. 登录系统

1. 打开前端页面或 Swagger 接口页面。
2. 使用测试账号接口查看可用账号。
3. 输入用户名和密码登录，例如 `admin / 123456`。
4. 登录成功后进入首页或获取接口 token。

## 2. 查看房态

- 进入房态管理页面。
- 可按楼层或房态筛选房间。
- 房态颜色和状态文本应与后端返回一致。
- 可对可操作房间执行维修、恢复空净等状态变更。

## 3. 办理预订

1. 进入预订管理页面。
2. 填写姓名、手机号、房型、入住日期和退房日期。
3. 提交预订。
4. 系统返回预订记录，目标房间状态变为已预订。

## 4. 办理入住

- 预订入住：选择已有预订并确认入住。
- Walk-in 入住：直接选择可用房型或房间，录入住客信息。
- 入住成功后系统创建客史、入住记录和账单。

## 5. 消费入账和退房

1. 在账单管理中选择目标账单。
2. 如有额外服务消费，追加费用项目。
3. 进入退房结算，确认应收金额和付款状态。
4. 退房成功后房间进入脏房状态，并生成清洁任务。

## 6. 客房清洁

- 客房服务员查看待处理任务。
- 将任务从待处理更新为进行中。
- 完成清洁并通过检查后，房态恢复为空净。

## 7. 报表查看

- 入住率报表展示总房数、在住数和入住率。
- 营收报表展示总营收、服务收入和账单数。
- 报表数据来自当前数据库状态。

## 8. 常见问题

| 问题 | 处理方式 |
| --- | --- |
| 登录失败 | 确认已导入 `seed.sql`，并使用测试账号 |
| 接口返回数据库错误 | 确认 MySQL 启动、密码正确、数据库已初始化 |
| 前端访问不到后端 | 检查 `BASE_URL` 和后端端口是否一致 |
| APP 端配置 JSON 校验失败 | 使用 HBuilderX 或 JSONC 解析；严格 JSON 工具需移除注释 |
